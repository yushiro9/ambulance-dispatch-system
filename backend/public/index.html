Reminder: These vehicles are for scheduled treatment/procedure transport only. For medical emergencies, call the facility's emergency line directly — do not use this booking system.
Scheduling rule: each ambulance handles one patient at a time. The system enforces a strict, non-negotiable 3-hour gap after every transport before that same vehicle can be booked again — well above the 2-hour minimum — for both Scheduler and Super Admin accounts. No booking that violates this can be saved or approved.
